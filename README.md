# Bluetooth-RC-car
In this project, I tried to make a Blutooth controlled RC car. I used the following components in this project:
1. Arduino Uno
2. L298n Motor Driver
3. HC-05 Bluetooth Transreciever Module
4. BO Motors
5. Li-ion Battery
